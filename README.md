# 🎨 GERADOR DE CSS IA

Um projeto que utiliza **Inteligência Artificial** para auxiliar desenvolvedores na criação de estilos CSS de forma rápida, prática e personalizada.  
O objetivo é simplificar o processo de estilização, oferecendo sugestões inteligentes e automáticas para layouts modernos e responsivos.

---

## 🚀 Funcionalidades
- Geração automática de código CSS com base em descrições de texto.
- Sugestões de estilos responsivos (Flexbox, Grid).
- Personalização de cores, fontes e bordas.
- Interface simples e intuitiva para desenvolvedores.
